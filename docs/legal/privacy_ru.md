# Политика конфиденциальности — «ИИ Бот №1»

1. **Какие данные обрабатываем.** Telegram `user_id`, username, язык интерфейса,
   настройки, баланс и история транзакций/генераций. Промпты и загруженные файлы
   обрабатываются для выполнения запроса.
2. **Зачем.** Предоставление сервиса, биллинг, антифрод, поддержка.
3. **Контекст диалога** хранится временно в Redis (rolling) и удаляется командой
   /deletecontext или по TTL.
4. **Передача третьим лицам.** Промпты передаются провайдерам нейросетей
   (OpenAI, Google, Anthropic и др.) и платёжным шлюзам — строго для исполнения
   запроса/платежа.
5. **Хранение.** Данные аккаунта и транзакций хранятся в PostgreSQL; резервные
   копии — по политике ретеншена. Результаты генераций — в объектном хранилище.
6. **Права пользователя.** Запрос на удаление данных — через поддержку.
7. **Безопасность.** Доступ к админ-панели — по IP allow-list, 2FA и журналу
   аудита; PII маскируется в логах.

> Базовый шаблон — замените согласованной с юристом версией (учтите 152-ФЗ/РКН и
> требования платёжных провайдеров) перед запуском.
