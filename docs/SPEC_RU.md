# ТЗ — Модернизация ChatGPTFlashBot (RU)

> Источник: серия решений владельца (опрос через UI). Реализуется **волнами по
> приоритетам** — объём многомесячный, не «за один проход». Почти всё должно
> управляться из админ-панели **без переустановки** (live-конфигурация).

## 0. Сквозные принципы
- **Live-управление из админки** всем: цены, лимиты, модели, сервисы, тексты,
  инлайн-кнопки, все фичи/механики (вкл/выкл + параметры).
- **Мульти-бот (white-label):** 2–3+ ботов на одном движке, **данные раздельные
  по боту** (мульти-тенант).
- **Масштаб до 3 млн+** (баланс скорости/надёжности/стоимости).
- **Хостинг:** всё на одном **VPS** (docker-compose), с готовностью разнести
  Postgres/Redis и добавить серверы при росте.
- Завершить **diamonds → credits**; валюта — значок **✨ везде**.

## 1. ФУНДАМЕНТ — движок live-конфигурации
Новый сервис (БД-таблица `pricing` как KV + Redis-кэш), читает/пишет из админки:
- **Цены:** подписки (premium/premium_x2 × 1/3/6/12), паки (image/video/music),
  пакеты кредитов, аватар, награда реферала, цена песни, документы.
- **Лимиты:** free_text_weekly, free_miniapp_weekly, premium_daily, premium_x2_daily.
- **Per-model:** стоимость (cost) + платность (premium) — из БД, не из кода.
- **Сервисы** (фото/видео/музыка/документы): вкл-выкл + цена.
- **Тексты/бренд:** название, приветствие, поддержка, ключевые тексты — 8 языков.
- Страница «Цены» — удобные поля вместо JSON, реально применяется.

## 2. AI-роутинг / OmniRoute / LiteLLM
- **+LiteLLM** равноправно рядом с OmniRoute (оба OpenAI-совместимые), в
  docker-compose, «всё что поддерживает».
- Для **каждой модели** вручную выбирать шлюз (OmniRoute/LiteLLM/прямой API).
- **Объединить** статичный список моделей и каталог БД → одно место (из админки).
- Ключи провайдеров для LiteLLM — и в админке, и в LiteLLM.
- «Проверить подключение», расход+лимиты трат, логи запросов/ошибок,
  latency/uptime, **балансировка по весам**, наценка/себестоимость, расходы
  LiteLLM в админке, **импорт/экспорт конфига**.
- **Управление контейнерами роутеров** (конфиг+перезапуск, старт/стоп, встроенные
  панели) — **только superadmin, ограниченно** (риск: доступ к серверу).

## 3. Telegram-бот
Стриминг ответов · Markdown+код (безопасный рендер) · inline-режим · память
**5–10 пар** · `/s→/search` (улучшить, из админки) · `/ava→/avatar` (подключить,
цена из админки) · `/invite` · готовые роли/персоны · редактирование фото в чате ·
красивая воронка покупки · приветственное медиа в /start · **инлайн-кнопки
настраиваются из админки (конструктор)** · работа в группах · уведомление о
готовности генерации.

## 4. Mini App
Значок ✨ · профиль: карточка аккаунта, статистика, управление подпиской (история
покупок не нужна; 4 вкладки оставить) · кнопка «Скачать» к результату · библиотека
пресетов в стиле Higgsfield (параметры из админки) · публичная галерея с
модерацией · адаптив/перформанс (сделано).

## 5. Генерация
Фото (6) и видео (8) + 74 Kling-эффекта — вкл-выкл+цена из админки, +добавлять
сервисы, Kling-эффекты редактируемы · музыка/документы — параметры из админки ·
фото-инструменты (Face Swap/Upscale/Аватары) подключить · 2–4 варианта результата ·
кнопка «HD/Улучшить» · картинки в полном качестве (файл) · срок хранения из
админки · прогресс-бар.

## 6. Платежи / монетизация
5 шлюзов (Stars/СБП-Tribute/ЮКасса/Stripe/CryptoBot) · **чеки 54-ФЗ** (ЮКасса) ·
цены в ₽/$ · **автопродление Premium** · промо (приветственный бонус, кэшбэк,
акции по времени, бонус за 1-ю покупку) · подарки другу · **агентская программа**
(выплаты кредитами И деньгами) · реклама бесплатным (Premium без рекламы) ·
**VIP/уровни** · реф-антифрод из админки. НЕ делаем: бандлы, своя защита платежей.

## 7. Уведомления / вовлечение
Авто-уведомления (окончание Premium, низкий баланс, бонус, возврат неактивных) ·
написать юзеру из админки + инбокс поддержки + кнопка «Поддержка» · конкурсы ·
автопостинг в канал · оценки 👍/👎 + жалобы · отслеживание источников трафика.

## 8. Админ-панель
- **Организация:** сворачиваемые группы, поиск (Cmd+K), дашборд «требует внимания»
  + быстрые действия.
- **Безопасность:** ключи→superadmin, **2FA обязательна** (admin+superadmin),
  управление админами из панели.
- **Новые разделы:** System Health · Очередь+ретрай · Экспорт CSV · аудит-фильтры ·
  бэкап/восстановление (superadmin) · просмотр логов · **конструктор кнопок** ·
  **редактор локализации (8 языков)** · управление роутерами.
- **Метрики/аналитика:** выручка за период, DAU, конверсия+ARPU, воронка,
  источники трафика, retention, авто-отчёты по расписанию, контент-аналитика,
  экспорт CSV/Excel.
- **Операционка:** режим техработ · CRM-заметки/теги · анти-абьюз (вкл-выкл) ·
  приоритет очереди для Premium.

## 9. НЕ делаем (зафиксировано)
Свой публичный API · удаление данных по запросу · ватермарка · бандлы · онбординг ·
бесплатный «крючок» новым · белый список модерации · отдельная защита платежей ·
лендинг/юр./тема оформления.

## 10. Приоритеты (волны реализации)
1. **Фундамент:** движок live-конфигурации (цены/лимиты/модели/сервисы/тексты из БД).
2. **AI-роутинг:** объединить модели, per-model шлюз, LiteLLM в compose + роль,
   мониторинг/метрики/расход.
3. **Мульти-бот + масштаб 3 млн** (раздельные данные по боту).
4. **Монетизация:** промо, автопродление, чеки, цены ₽/$, VIP, агенты, подарки.
5. **Вовлечение:** уведомления, источники трафика, конкурсы, автопостинг, оценки.
6. **Контент/UX:** сервисы вкл-выкл+цены, фото-инструменты, варианты/HD, пресеты,
   галерея.
7. **Бот:** стриминг, Markdown, inline, память, /search, /avatar, /invite, роли,
   редакт фото, группы.
8. **Админка:** организация+поиск+дашборд, новые разделы, 2FA-обязательность,
   ключи→superadmin, управление админами, конструктор кнопок, локализация.
9. **Прочее:** CRM-заметки, анти-абьюз, техработ, приоритет очереди, аналитика.

## 11. Предупреждения / координация
- Объём огромный → реализуется волнами, не сразу.
- Управление контейнерами из веб-админки = доступ к серверу → строго superadmin +
  изоляция.
- Параллельная сессия строит часть фич (daily_bonus, CryptoBot, provider-keys, 2FA,
  рассылки, рефералы) — перед мерджем свести ветки и устранить коллизию миграций
  (два `0007` → один head через `alembic merge heads` или переименование в `0008`).
- Уже сделано: UI-адаптив/перформанс; provider-keys/2FA/moderation-words/
  CryptoBot/daily_bonus (2-я сессия).
